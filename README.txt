Controls:

Movement:
Use the A key to move the player character left.
Use the D key to move the player character right.

Pause Menu:
Press the Escape key during gameplay to pause the game and access the pause menu.

While in the pause menu:
Press Escape again to resume the game.
Press Q to quit the game.

Game Over Menu:
When the player character collides with a bomb, the game ends, and the game over menu appears.

While in the game over menu:
Press R to restart the game.
Press Q to quit the game.
